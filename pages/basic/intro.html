SQLとは

SQL（Structured Query Language）は、 データベースを操作するための言語です。

📌 データベースとは？

データを保存・管理する仕組みのことです。

例えば、以下のような情報が保存されています。

  • ユーザー情報
  • 商品情報
  • 注文履歴

📌 基本用語

用語 意味
テーブル データを保存する表
カラム（列） nameやageなどの項目
レコード（行） 1件分のデータ

📌 SQLでできること

  • データの取得（SELECT）
  • データの追加（INSERT）
  • データの更新（UPDATE）
  • データの削除（DELETE）

📌 CRUD操作とは？

操作 意味 SQL
Create 追加 INSERT
Read 取得 SELECT
Update 更新 UPDATE
Delete 削除 DELETE

💡 SQLを学ぶメリット

  • データの流れを理解できる
  • バックエンド開発に活かせる
  • 多くの現場で使用されている
