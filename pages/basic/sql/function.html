関数

SQLでは、データの集計や計算を行うための関数を使用できます。

📌 COUNT関数

SELECT COUNT(*)
FROM users;

→ レコード数を取得する

📌 SUM関数

SELECT SUM(price)
FROM products;

→ priceの合計値を取得する

📌 AVG関数

SELECT AVG(age)
FROM users;

→ ageの平均値を取得する

📌 MAX関数

SELECT MAX(age)
FROM users;

→ ageの最大値を取得する

📌 MIN関数

SELECT MIN(age)
FROM users;

→ ageの最小値を取得する

💡 ポイント

  • 関数はSELECT句でよく使用される
  • GROUP BYと組み合わせることが多い
  • 集計処理で頻繁に使用される

📝 問題①（基礎）

productsテーブルのprice合計を取得するSQLを書いてみましょう。

答えを見る 
SELECT SUM(price)
FROM products;

📝 問題②（応用）

usersテーブルのdepartmentごとの平均年齢を取得するSQLを書いてみましょう。

答えを見る 
SELECT department, AVG(age)
FROM users
GROUP BY department;
