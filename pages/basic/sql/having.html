HAVING句

HAVING句は、GROUP BYでグループ化した後のデータに条件を指定するための構文です。

📌 GROUP BYだけの場合

SELECT department, COUNT(*)
FROM users
GROUP BY department;

→ 部署ごとの人数を取得する

📌 HAVINGを使うと？

SELECT department, COUNT(*)
FROM users
GROUP BY department
HAVING COUNT(*) >= 2;

→ 人数が2人以上の部署だけ取得する

💡 WHEREとの違い

  • WHERE：グループ化する前に条件を指定する
  • HAVING：グループ化した後に条件を指定する

📝 問題①（基礎）

部署ごとの人数を取得し、その中から「人数が3人以上」の部署だけ表示するSQLを書いてみましょう。

答えを見る 
SELECT department, COUNT(*)
FROM users
GROUP BY department
HAVING COUNT(*) >= 3;

📝 問題②（応用）

部署ごとの平均年齢を取得し、「平均年齢が30以上」の部署だけ表示するSQLを書いてみましょう。

答えを見る 
SELECT department, AVG(age)
FROM users
GROUP BY department
HAVING AVG(age) >= 30;
