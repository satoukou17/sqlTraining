論理演算子

論理演算子は、複数の条件を組み合わせるために使用する演算子です。

📌 主な論理演算子

演算子 意味
AND 両方の条件を満たす
OR どちらかの条件を満たす
NOT 条件に一致しない

📌 ANDの例

SELECT *
FROM users
WHERE age >= 20
AND department = '開発';

→ ageが20以上、かつdepartmentが「開発」のデータを取得する

📌 ORの例

SELECT *
FROM users
WHERE department = '営業'
OR department = '開発';

→ 営業または開発のデータを取得する

📌 NOTの例

SELECT *
FROM users
WHERE NOT department = '営業';

→ 営業以外のデータを取得する

💡 ポイント

  • ANDは「かつ」
  • ORは「または」
  • 複数条件を扱う際によく使用される

📝 問題①（基礎）

usersテーブルから「ageが25以上」かつ「departmentが開発」のデータを取得するSQLを書いてみましょう。

答えを見る 
SELECT *
FROM users
WHERE age >= 25
AND department = '開発';

📝 問題②（応用）

usersテーブルから「departmentが営業または開発」のデータを取得するSQLを書いてみましょう。

答えを見る 
SELECT *
FROM users
WHERE department = '営業'
OR department = '開発';
