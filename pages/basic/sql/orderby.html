ORDER BY句

ORDER BY句は、取得したデータを並び替えるための構文です。

📌 ORDER BYを使わない場合

SELECT name, age
FROM users;

→ データの順番は保証されない

📌 ORDER BYを使うと？

SELECT name, age
FROM users
ORDER BY age;

→ ageの小さい順に並び替える

📌 DESCを使う場合

SELECT name, age
FROM users
ORDER BY age DESC;

→ ageの大きい順に並び替える

💡 ポイント

  • ORDER BYの後に並び替え対象の列を書く
  • ASC：昇順（小さい順）
  • DESC：降順（大きい順）

📝 問題①（基礎）

usersテーブルを「ageの昇順」で並び替えるSQLを書いてみましょう。

答えを見る 
SELECT name, age
FROM users
ORDER BY age;

📝 問題②（応用）

usersテーブルを「ageの降順」で並び替えるSQLを書いてみましょう。

答えを見る 
SELECT name, age
FROM users
ORDER BY age DESC;
