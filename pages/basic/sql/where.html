WHERE句

WHERE句は、データを条件で絞り込むための構文です。

📌 まずは復習（SELECT + FROM）

SELECT name, age
FROM users;

→ 全てのデータを取得する

📌 WHEREを使うと？

SELECT name, age
FROM users
WHERE age >= 20;

→ ageが20以上のデータだけ取得する

💡 ポイント

  • WHERE句に条件を書く
  • 比較演算子（=, >=, <= など）を使用する
  • ANDやORを使って複数条件も指定できる

📝 問題①（基礎）

usersテーブルから「ageが30以上」のデータを取得するSQLを書いてみましょう。

答えを見る 
SELECT name, age
FROM users
WHERE age >= 30;

📝 問題②（応用）

usersテーブルから「ageが20以上かつ30以下」のデータを取得するSQLは？

答えを見る 
SELECT name, age
FROM users
WHERE age >= 20
AND age <= 30;
