BETWEEN句

BETWEEN句は、指定した範囲のデータを取得するための構文です。

📌 BETWEENを使わない場合

SELECT name, age
FROM users
WHERE age >= 20
AND age <= 30;

→ ageが20以上30以下のデータを取得する

📌 BETWEENを使うと？

SELECT name, age
FROM users
WHERE age BETWEEN 20 AND 30;

→ ageが20〜30のデータを取得する

💡 ポイント

  • BETWEENは範囲検索で使用する
  • 「下限 AND 上限」の順で書く
  • WHERE句と一緒によく使われる

📝 問題①（基礎）

usersテーブルから「ageが25〜35」のデータを取得するSQLを書いてみましょう。

答えを見る 
SELECT name, age
FROM users
WHERE age BETWEEN 25 AND 35;

📝 問題②（応用）

productsテーブルから「priceが1000〜5000」のデータを取得するSQLを書いてみましょう。

答えを見る 
SELECT product_name, price
FROM products
WHERE price BETWEEN 1000 AND 5000;
