比較演算子

比較演算子は、値を比較するために使用する演算子です。 主にWHERE句で使用されます。

📌 主な比較演算子

演算子 意味
= 等しい
!= 等しくない
> より大きい
< より小さい
>= 以上
<= 以下

📌 使用例

SELECT name, age
FROM users
WHERE age >= 20;

→ ageが20以上のデータを取得する

📌 「等しくない」の例

SELECT name
FROM users
WHERE department != '営業';

→ 営業以外のデータを取得する

💡 ポイント

  • WHERE句でよく使用される
  • 数値だけでなく文字列にも使用できる
  • 条件分岐の基本となる

📝 問題①（基礎）

usersテーブルから「ageが25以上」のデータを取得するSQLを書いてみましょう。

答えを見る 
SELECT *
FROM users
WHERE age >= 25;

📝 問題②（応用）

usersテーブルから「departmentが開発ではない」データを取得するSQLを書いてみましょう。

答えを見る 
SELECT *
FROM users
WHERE department != '開発';
