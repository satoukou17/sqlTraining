DELETE文

DELETE文は、テーブルのデータを削除するための構文です。

📌 DELETE文の基本構文

DELETE FROM users
WHERE name = '田中';

→ 「田中」のデータを削除する

📌 実行前のイメージ

name age
田中 25
佐藤 30

📌 実行後のイメージ

name age
佐藤 30

💡 ポイント

  • DELETE FROM の後にテーブル名を書く
  • WHERE句で削除対象を指定する
  • WHERE句を書かないと全件削除されるので注意

📝 問題①（基礎）

usersテーブルから「佐藤」のデータを削除するSQLを書いてみましょう。

答えを見る 
DELETE FROM users
WHERE name = '佐藤';

📝 問題②（応用）

productsテーブルから「priceが1000未満」のデータを削除するSQLを書いてみましょう。

答えを見る 
DELETE FROM products
WHERE price < 1000;
