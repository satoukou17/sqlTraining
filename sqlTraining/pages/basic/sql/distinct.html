DISTINCT句

DISTINCT句は、重複したデータを除外するための構文です。

📌 DISTINCTを使わない場合

SELECT department
FROM users;

→ 同じdepartmentが重複して表示される

📌 DISTINCTを使うと？

SELECT DISTINCT department
FROM users;

→ 重複を除外してdepartmentを表示する

💡 ポイント

  • DISTINCTは重複データを除外する
  • SELECTの直後に書く
  • GROUP BYでも似たことはできるが、DISTINCTの方がシンプル

📝 問題①（基礎）

usersテーブルから重複しないdepartmentを取得するSQLを書いてみましょう。

答えを見る 
SELECT DISTINCT department
FROM users;

📝 問題②（応用）

productsテーブルから重複しないcategoryを取得するSQLを書いてみましょう。

答えを見る 
SELECT DISTINCT category
FROM products;
