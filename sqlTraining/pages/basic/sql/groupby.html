GROUP BY句

GROUP BY句は、同じ値ごとにデータをグループ化するための構文です。

📌 GROUP BYを使わない場合

SELECT department
FROM users;

→ 同じ部署名が重複して表示される

📌 GROUP BYを使うと？

SELECT department
FROM users
GROUP BY department;

→ departmentごとにグループ化される

📌 COUNT関数と組み合わせる

SELECT department, COUNT(*)
FROM users
GROUP BY department;

→ 部署ごとの人数を取得できる

💡 ポイント

  • GROUP BYの後にグループ化したい列を書く
  • COUNTやSUMなどの関数とよく組み合わせる
  • 集計処理でよく使われる

📝 問題①（基礎）

usersテーブルをdepartmentごとにグループ化するSQLを書いてみましょう。

答えを見る 
SELECT department
FROM users
GROUP BY department;

📝 問題②（応用）

部署ごとの人数を取得するSQLを書いてみましょう。

答えを見る 
SELECT department, COUNT(*)
FROM users
GROUP BY department;
