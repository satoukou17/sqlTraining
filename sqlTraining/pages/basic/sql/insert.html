INSERT文

INSERT文は、テーブルに新しいデータを追加するための構文です。

📌 INSERT文の基本構文

INSERT INTO users (name, age)
VALUES ('田中', 25);

→ usersテーブルにデータを追加する

📌 実行後のイメージ

name age
田中 25

💡 ポイント

  • INSERT INTO の後にテーブル名を書く
  • VALUES に追加する値を書く
  • 列名と値の順番を合わせる

📝 問題①（基礎）

usersテーブルに「佐藤, 30」を追加するSQLを書いてみましょう。

答えを見る 
INSERT INTO users (name, age)
VALUES ('佐藤', 30);

📝 問題②（応用）

productsテーブルに「PC, 120000」を追加するSQLを書いてみましょう。

答えを見る 
INSERT INTO products (product_name, price)
VALUES ('PC', 120000);
