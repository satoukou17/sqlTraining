LIKE句

LIKE句は、文字列をあいまい検索するための構文です。

📌 LIKEを使わない場合

SELECT name
FROM users
WHERE name = '田中';

→ 「田中」と完全一致するデータだけ取得する

📌 LIKEを使うと？

SELECT name
FROM users
WHERE name LIKE '%田%';

→ 「田」を含むデータを取得する

📌 ワイルドカード

  • % ：0文字以上の任意の文字列
  • _ ：任意の1文字

📌 LIKEの例

-- 「田」で始まる
LIKE '田%'

-- 「田」で終わる
LIKE '%田'

-- 「田」を含む
LIKE '%田%'

💡 ポイント

  • LIKEはWHERE句と一緒によく使う
  • 文字列検索で頻繁に使用される
  • %を使うと部分一致検索ができる

📝 問題①（基礎）

usersテーブルから「山」を含むnameを取得するSQLを書いてみましょう。

答えを見る 
SELECT name
FROM users
WHERE name LIKE '%山%';

📝 問題②（応用）

usersテーブルから「田」で始まるnameを取得するSQLを書いてみましょう。

答えを見る 
SELECT name
FROM users
WHERE name LIKE '田%';
