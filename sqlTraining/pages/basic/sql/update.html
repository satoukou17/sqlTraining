UPDATE文

UPDATE文は、既存のデータを更新するための構文です。

📌 UPDATE文の基本構文

UPDATE users
SET age = 30
WHERE name = '田中';

→ 「田中」のageを30に更新する

📌 実行前のイメージ

name age
田中 25

📌 実行後のイメージ

name age
田中 30

💡 ポイント

  • UPDATEの後に更新対象のテーブルを書く
  • SETで更新内容を指定する
  • WHERE句を書かないと全件更新されるので注意

📝 問題①（基礎）

usersテーブルの「佐藤」のageを35に更新するSQLを書いてみましょう。

答えを見る 
UPDATE users
SET age = 35
WHERE name = '佐藤';

📝 問題②（応用）

productsテーブルの「PC」のpriceを150000に更新するSQLを書いてみましょう。

答えを見る 
UPDATE products
SET price = 150000
WHERE product_name = 'PC';
